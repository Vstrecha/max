# Frontend Service

Vue.js frontend приложение для системы управления мероприятиями.

## Описание

Frontend сервис предоставляет пользовательский интерфейс для работы с системой управления мероприятиями. Приложение разработано как Max Mini App и интегрируется с Max платформой.

## Технологии

- **Vue.js 3** - прогрессивный JavaScript фреймворк
- **TypeScript** - типизированный JavaScript
- **Pinia** - state management
- **Vue Router** - маршрутизация
- **Vite** - сборщик и dev-сервер
- **Vant** - UI компоненты для мобильных приложений

## Структура проекта

```
src/
├── components/      # Vue компоненты
├── views/          # Страницы приложения
├── stores/         # Pinia stores
├── controllers/    # API контроллеры
├── types/          # TypeScript типы
├── router/         # Конфигурация роутера
└── utils/          # Утилиты
```

## Основные функции

- Просмотр и создание мероприятий
- Управление профилем пользователя
- Система друзей
- Регистрация на мероприятия
- Сканирование QR-кодов для отметки посещения
- Загрузка и управление файлами (аватары, фото мероприятий)

## Запуск

### Через Docker Compose

```bash
docker-compose up frontend-builder
```

### Локальная разработка

```bash
cd frontend
npm install
npm run dev
```

### Сборка для продакшена

```bash
npm run build
```

## Переменные окружения

- `VITE_HOST_API_URL` - URL backend API

## Конфигурация

Файл `vite.config.ts` содержит конфигурацию сборщика и dev-сервера.

## Сборка

Frontend собирается в Docker контейнере и результат сохраняется в volume `frontend-dist`, который монтируется в nginx для раздачи статических файлов.
